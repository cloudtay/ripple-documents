## 介绍

ripple所有的支持库都处于`Co`命名空间下,并根据它所属的模块进行了划分,目前支持库有以下几个模块:

- `Co\Net` 网络操作模块
- `Co\IO` 流资源操作模块
- `Co\System` 系统操作模块
- `Co\Util` 工具模块
- `Co\Plugin` 插件模块
- `Co\Parallel` 多线程模块

你可以通过以下方式访问支持库并使用,如

```php
use \Co\IO;

$fileUtil = IO::File();
$content = $fileUtil->getContent('file.txt')->await();
```

也支持链式调用如

```php
$channel = \Co\IO::Channel()->open('master');
$channel->send('hello world');
```
